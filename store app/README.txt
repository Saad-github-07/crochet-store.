Quick start
-----------
1) Open script.js and set your WhatsApp number:
   const WHATSAPP_NUMBER = "+91YOURNUMBER";

2) (Optional) Add prices by changing `price: null` to a number (e.g., 199).

3) Deploy:
   - Netlify: Drag the whole folder onto https://app.netlify.com/drop
   - GitHub Pages: Push to a repo and enable Pages (root or /docs)

Files
-----
- index.html  : main page
- styles.css  : pastel, responsive design
- script.js   : products + WhatsApp order links
- /assets     : product images (your photos are already placed)

Branding
--------
Change site title/logo text in index.html:
  <h1>Bloom &amp; Pearl</h1>

Change colors in styles.css (look for :root variables).
